# Employee Management System

A full-stack web application built using **React, Tailwind CSS, Django, Django REST Framework, and SQL**.

## Features

### Admin
- Secure login
- View all employees
- Dynamic dashboard
- Role-based access

### Employee
- Secure login
- View and update profile
- Protected routes

## Tech Stack

Frontend:
- React.js
- Tailwind CSS
- Axios

Backend:
- Django
- Django REST Framework
- SQLite / MySQL

## Security
- Custom user model
- Role-based permissions
- Backend-enforced authorization

## Run Locally

### Backend
```bash
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python manage.py runserver
```
##Frontend
```bash
cd frontend
npm install
npm start
```
